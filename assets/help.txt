•• GLOBAL

'Tab'    #switch view Desktop/Patterns or Themes/Fav-Themes
'Up'     #previous item
'Down'   #next item
'Left'   #focus left panel
'Right'  #focus right panel
'o'      #show/hide extras popup
'l'      #show/hide logs
'c'      #toggle colors preview
'q'      #quit
'h'      #show/hide this popup

•• PATTERNS

'i'      #toggle invert pattern's colors
'p'      #edit pattern's post-script in $VISUAL env var
'e'      #edit pattern's file in $VISUAL env var

•• DESKTOPS

'Enter'  #apply desktop
